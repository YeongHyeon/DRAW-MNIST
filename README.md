# DRAW: Deep Recurrent Attentive Writer

## Requirements
* Python 3.5.2  
* Tensorflow 1.4.0  
* Numpy 1.13.3  
* Scipy 1.2.0  
* Matplotlib 3.0.2  

## Usage
Just run the following command in the terminal.
```
$ python run.py
$ python run.py --help // for confirming the option
```
